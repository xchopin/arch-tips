# Arch Tips
Some tips I use at work for my Arch Linux installations


## Essential packages to download
- `pacman -Sy yaourt`
- `yaourt sublime-text`
- `yaourt atom`
- `yaourt google-chrome`
- `yaourt spotify`
- `yaourt openconnect`
- `yaourt owncloud`
- `yaourt gtk-arc-flatabulous-theme-git`
- `yaourt mariadb`
